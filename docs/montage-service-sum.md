# Передача суммы услуг в монтаж

## Поле сделки

- Название: **Сумма услуг**
- ID пользовательского поля Bitrix24: **2788**
- Внутренний код: **`UF_CRM_SERVICE_SUM`**
- Внешний код: **`B24_APP_DEAL_SERVICE_SUM`**
- Тип: **Деньги**
- Валюта: **RUB**

Приложение считает сумму только по строкам текущего рабочего состава сделки, отмеченным как услуги/работы:

`Σ (итоговая цена после скидки × количество)`

При отсутствии услуг записывается `0 RUB`. Поле пересчитывается при изменении рабочего состава, выборе варианта КП и возврате позиции.

## Настройка бизнес-процесса

В действии **«Создать элемент смарт-процесса Монтаж»** нужно установить:

- **Ручной расчёт суммы** — `Да`;
- **Валюта** — `Российский рубль`;
- **Сумма** — значение поля сделки `UF_CRM_SERVICE_SUM` («Сумма услуг»).

Поле смарт-процесса **«Оплата за выезд (если платно)»** для этой задачи не используется: оно означает отдельную оплату инженеру, а не стоимость работ по сделке.
