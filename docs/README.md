## Главное за 30 секунд

- **Что это:** локальное приложение Битрикс24 для портала `umniydom.bitrix24.ru` — свои вкладки и окна поверх Б24 (товары сделки, база товаров, инвентаризация, реализации, отчёты).
- **Стек:** Node.js + TypeScript + Fastify (бэкенд), React + Vite (фронт в iframe), монорепо npm workspaces.
- **Хостинг:** один корп-VPS `201.51.12.57` (Ubuntu, 24/7) — ядро ERPNext, backend, дверь nginx+TLS (`201.51.12.57.sslip.io`), синк, бэкапы. Всё в [network.md](network.md). Поднять/восстановить — [runbook.md](runbook.md), здоровье — `status`, аварийка — [SOS.md](SOS.md). Источник правды: код в GitHub, данные в бэкапе.
- **Стратегия:** складской учёт выезжает из Б24 в headless ERPNext («покрывало», см. sklad-vynos.md); Б24 остаётся CRM.
- **Правило проекта:** не писать код без обсуждения; не удалять сущности на портале (зачистка тестов — за Сергеем).
